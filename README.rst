.. _description:

Features:

    - One single module;
    - No requirements;
    - Tiny interface;
    - Supports variable interpolation;

.. _requirements:

Requirements
=============

- python (2.6, 2.7, 3.2, 3.3)


.. _installation:

Installation
=============

**Inirama** could be installed using pip: ::

    pip install inirama


.. _quickstart:

Quickstart
==========

::

    from inirama import Namespace

    ns = Namespace()
    ns.read('config.ini')

    print ns['section']['key']

    ns['other']['new'] = 'value'
    ns.write('new_config.ini')


.. _interpolation:

Interpolation
-------------
::

    from inirama import InterpolationNamespace

    ns = InterpolationNamespace()
    ns.parse("""
        [main]
        test = value
        foo = bar {test}
        more_deep = wow {foo}
    """)
    print ns['main']['more_deep']  # wow bar value
